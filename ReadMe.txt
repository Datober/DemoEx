
ООО «Стройматериалы»  - магазин по продаже строительных материалов в Сыктывкаре. 
Приложение находится в открытом доступе на гитхабе и может быть установлено практически на любой персональный компьютер
Для установки требуется открыть GitHub и найти нужный репозиторий, нажав на кнопку Load загрузить архив и распаковать его на рабочем столе
При открытии приложения требуется заполнить поля логина и пароля, или же зайти в приложение в качестве гостя. 
При авторизированном входе у клиента/менеджера появляется возможность выбирать, удалять товар. Администратор же может его и редактировать
